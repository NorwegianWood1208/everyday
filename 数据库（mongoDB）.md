##使用方法：
	1.	需要安装node / mongoDB    https://www.mongodb.com/ 安装

	启动数据库:
		找到C:\Program Files\MongoDB\Server\3.4\bin

		.\mongod --dbpath=数据库存放的位置 --port=27017 
	
	2.	使用npm install 安装依赖项

	3.	npm run dev  运行服务

	4.	在http://localhost:88 访问

	npm i nrm -g

	nrm test  查看哪个速度快

	nrm use cnpm  切换镜像
